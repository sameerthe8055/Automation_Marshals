Project Description
AnyLangTranslate is a UiPath-based automation project that translates text from any language to a specified target language in real-time. It also generates a summary report of all the translations performed.

Folder Contents
Project Folder: Contains the UiPath project files.
Presentation: PowerPoint presentation on the project.
